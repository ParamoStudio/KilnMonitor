Guide
Setup & usage

Páramo connects to your kiln's ESP32 over MQTT. Below you'll find setup instructions, connection details, and usage notes. Full documentation coming soon.

01
Configure your ESP32
Flash the Páramo firmware and set your WiFi credentials. The ESP32 will publish temperature data to the MQTT broker at regular intervals. — placeholder
02
Set up a HiveMQ broker
Create a free account at HiveMQ Cloud. Copy your cluster host, username, and password into the Connection panel in the sidebar. — placeholder
03
Connect & monitor
Hit Connect. The status pills in the header will turn green when both the MQTT server and kiln are reachable. Your firing data starts streaming. — placeholder
04
Set thresholds & firing mode
Use the Alarm Threshold card to receive an alert when a target temperature is reached. Select a firing mode to overlay a reference ramp curve on the chart. — placeholder
05
Export your session
When the firing is complete, export the chart as JSON for archiving or analysis. Start a new firing session from the Session card. — placeholder
