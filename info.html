Guide
Setup and usage

Páramo connects to your kiln monitor via an MQTT server. Here you'll find setup instructions, connection details, and usage notes. Full documentation for makers and technical users is on the GitHub page. If you have any doubts, feel free to use the contact page.

How to get started
01
Connect your thermocouple
Páramo uses a Type S thermocouple for accurate high-temperature readings. See the Thermocouples Guide for recommended models and wiring instructions.
02
Prepare your server and notifications
2.1
HiveMQ account
We chose this free MQTT server because it provides more than enough capacity without extra costs for the end user. Alternative self-hosted servers are possible for more technical users — see the Servers Guide.

Go to hivemq.com, create a free account, and set a username and password. You will also be given a cluster token. Save all three for the next step.
2.2
Set up notifications
We chose NTFY as the notification provider for its simplicity and free pricing.

Download the app on any device, then create a new topic. Give it a name that is not easily guessable — just for you.
Example topic name: StudioKiln-HXsbi93DHF
In the topic settings you will find the Topic URL. Save it for the next step.
Example URL: ntfy.sh/StudioKiln-HXsbi93DHF
03
Turn on your Páramo Kiln Monitor
On first boot, the device creates a Kiln Setup Wi-Fi portal. Connect to it from any phone or laptop to configure your credentials.
3.1
Choose your preferred Wi-Fi network and enter the password. We recommend the most stable network near where the device will live — close to the kiln.
3.2
Enter the HiveMQ credentials you set up in step 2.1.
3.3
Paste the NTFY topic URL you created in step 2.2.
04
You're done ◉
Your Páramo Kiln Monitor now has everything it needs to communicate with you over the internet. You will receive temperature alerts and fault notifications directly on your device.
05
Web app setup and usage
5.1
Open paramostudio.github.io/KilnMonitor. On mobile, we suggest adding it to your home screen and using it as a web app.
5.2
This app relies on local cache. If you clear it, some configurations will be lost. Export them first using the toolbar — they can be re-imported on any device.
5.3
Add a kiln
On first open there is a blank "Studio Kiln" entry. Open the Connection panel under it and enter your HiveMQ credentials to link the device with the web app. You can rename or delete kilns using the buttons to the right, and add as many as you want — each one needs its own Páramo monitor.
5.4
Other tools
Kiln Power — bridges an MQTT relay to cut kiln power remotely in case of emergency. See the Kiln Power Guide. A solution for gas kilns is being studied.

Firing Mode — shows a reference ramp curve alongside your actual firing for easy comparison, and tags the session for later search in the Journal.

Cooldown Alarm — an alarm that triggers on the way down, once a firing has been detected and the kiln cools past a temperature you set. Useful to know when it is safe to start unloading. Adjustable from the Alarm Threshold card (150–550 °C).

Target Cone — note the Orton cone you are aiming for at the top of the Session card. It is purely a reference — it travels with the firing record and lets you search and group firings by cone in the Journal.

Kiln Type — choose your kiln type (electric, gas, wood or other) to control which power options are shown, or simply to tag the firing so you can filter by kiln type in the Journal.

Add Notes — write observations during or after a firing before exporting. Saved with the session in the Journal.

Export / Import / Reset — copy your configuration to other devices or reset all stored data.

Journal — a searchable database of all your firings, local or cloud. The cloud function uses Google Drive for convenience. If you'd rather keep things off Google, just export your sessions to a folder and load them manually in the Journal. A self-hosted solution is being explored, balancing user friction against the limited resources of an individual project.

Share as PNG — export any firing from the Journal as a printable greyscale image — chart, key data and notes laid out on an A4 sheet — to share with fellow potters or keep on paper.

Energy & Cost — set your price per kWh and currency to get a rough cost estimate per firing. This is an approximation only and should not be taken as a single source of truth — it cannot account for the complexity of a real electricity bill (tariff bands, standing charges, taxes, time-of-use rates).

Language — the app is available in several languages, selectable from the toolbar. Missing yours, or spotted a translation that could be better? You can contribute one — see the Add a Language Guide.

Having trouble? Contact us or check the Troubleshooting guide.

Why copyleft

A kiln should not be firstly perceived as a consumer product. It is a tool built to last decades, passed between studios, repaired by hand, adapted to the specific needs of the person using it. The electronics and software we build around kilns should be no different. Any action towards encouraging the right to repair, understand, recycle and upgrade is a step towards what we deserve as communities. In any context given.

Páramo was built on open resources, community knowledge, and freely shared prior work. It will remain open in return. Copyleft licenses were chosen deliberately — not by default — to prevent proprietary capture of derivatives, whether as closed software forks, closed web services, or proprietary hardware products.
